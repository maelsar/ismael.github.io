Fully Functional ReactJS Chat App

Project information

  • Category: Webapp
  • Project date: 11 April, 2023
  • Project URL: https://ploky-chat.firebaseapp.com/

This is a fully functional real-time chat app without the use of ready-made React libraries. Login Authentication, database and web hosting is powered by Firebase.

This webapp was made with ReactJS.
